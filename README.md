# dolus
A command-line interface to put effects on your virtual camera(OBS/Unity/v4l2loopback)



# Prerequisites

## Windows/MacOS

Python 3.7 or higher 

OBS 26 or Higher

## Linux 

Python 3.7 or higher 

v4l2loopback

# Installation

## Windows

```pip install dolus```

## Mac/Linux

```pip3 install dolus```


# Usage

Type dolus in the terminal/powershell
